这是一份自用的 Ubuntu 安装教程，不包含安装盘制作过程

设备：Windows 11 PC

版本：LTS 22.04

安装前请确保磁盘分区格式为 GUID 分区表(GPT)

Here we go~

  1. 关闭 BitLocker 和独显直连

  2. 压缩 Windows 分区

  3. 插上U盘，进入 BIOS：

    (1) 关闭 Secure Boot

    (2) 关闭 Intel RST

    (3) 设置U盘引导

  4. 安装 Ubuntu：

    (1) 在更新和其他软件中关闭安装 Ubuntu 时下载更新

    (2) 打开为图形或无线硬件......

    (3) 在安装类型中选择其他选项

    (4) 选中之前压缩的空闲分区，点击+，选择根挂载点

    (5) 选择 efi 分区的设备号，在下拉菜单中找到并选中

    (6) 点击现在安装

后续步骤较为简单，略

安装完成后记得在 BIOS 中设置 Ubuntu 引导
